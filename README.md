# README

## Project start

1. **Clone repository**

   ```bash
   git clone git@github.com:MateuszRajek/jobs-app.git
   cd jobs-app
   ```

2. **Instal dependencies**

   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run Expo**

   ```bash
   expo start
   ```

4. **Open application**

   - iOS: press `i`
   - Android: press `a`
